# אלגוריתמים — לראות בעיניים

מערכת אינטראקטיבית ללימוד הקורס "מבני נתונים ומבוא לאלגוריתמים". המטרה: **לראות את
האלגוריתם רץ** — אנימציות ואינטואיציה במקום קירות טקסט. ממשק עברי (RTL), קוד באנגלית.

השיעור הראשון שמומש: **שיעור 4 — מיון ערימה (Heapsort)**, עם ארבעה מצבים:
🎬 ויזואליזציה מודרכת · 🧪 ארגז חול · ✍️ תרגול אקטיבי · 📋 סיכום.

## הרצה מקומית

```bash
npm install
npm run dev      # http://localhost:5173
npm test         # בדיקות נכונות האלגוריתמים
npm run build    # בנייה ל-dist/
```

## ארכיטקטורה

הליבה היא **מנוע פריימים**: כל אלגוריתם הוא גנרטור דטרמיניסטי טהור שמפיק `Frame[]`
(ראו `src/engine/`). כל ארבעת המצבים צורכים את אותה רשימת פריימים. זהות נוסעת עם הערך
(`elementId`), לא עם התא — זה מה שמאפשר לאנימציית ההחלפה לעבוד.

- `src/engine/` — טיפוסים, `FrameBuilder`, חשבון אינדקסים (1-indexed, CLRS).
- `src/lectures/<id>/` — מודול שיעור עצמאי (אלגוריתמים + פסאודו-קוד + סיכום) המממש את `LectureModule`.
- `src/viz/` — התצוגה הכפולה מערך↔עץ (SVG + Framer Motion).
- `src/shell/`, `src/modes/` — מעטפת, נגן, ופאנלים — אגנוסטיים לשיעור.

**הוספת שיעור חדש** = ליצור `src/lectures/<id>/` שמייצא `LectureModule`, ולהוסיף שורה
ב-`src/app/registry.ts`. כל השאר (מצבים, נגן, RTL, deep-links, גזירת תרגול) מגיע חינם.

## פריסה

אתר סטטי. ב-Vercel: חברו את הרפו (root = `app/`), `vite build`. `vercel.json` כולל
SPA rewrite כדי ש-deep-links ישרדו רענון. ב-Netlify: `public/_redirects` עושה את אותו הדבר.
