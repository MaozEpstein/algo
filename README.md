## 🎯 המטרה

ללמד אלגוריתמים **בראייה ולא בקריאה**. במקום קירות טקסט — רואים את האלגוריתם רץ צעד-אחר-צעד,
עם הקוד שמתרחש לצידו, אנימציות שמראות מה קורה למבנה הנתונים, והאינטואיציה מאחורי כל מהלך.
הדגש הוא על **"לראות את זה"** — כך שגם מושגים מופשטים (סיבוכיות, רקורסיה, נוסחאות נסיגה)
הופכים מוחשיים. הממשק כולו בעברית (RTL); הפסאודו-קוד וה-Python נשארים באנגלית, כמו בקורס.

## ✨ היכולות

- **ויזואליזציה מודרכת** — נגן עם הרצה/השהיה/דילוג צעד-צעד וקיצורי מקלדת, עם **הדגשת שורת הקוד**
  הרצה (פסאודו-קוד + מתג ל-Python).
- **תצוגה כפולה** מערך ↔ עץ בינארי, עם אנימציות FLIP — הזהות נוסעת עם הערך, כך שהחלפות "זזות".
- **מוני עלות** (השוואות והחלפות) שמתעדכנים תוך כדי ריצה.
- **קלט ניתן לעריכה + דוגמאות מוכנות** (presets) של מקרי-קצה ומקרים גרועים בלחיצה.
- **הוכחות סיבוכיות** מובנות (טענה → צעדים → אינטואיציה), תגי "ראשי/עזר", וגרף קריאות.
- **מצב סיכום** עם KaTeX, טבלאות סיבוכיות וייצוא ל-PDF.
- **מבט-על בין-שיעורי** — טבלת השוואת מיונים ו"מירוץ אלגוריתמים" (שני אלגוריתמים על אותו קלט).
- **שיעורי "מסביר"** אינטראקטיביים לניתוח (ולא להרצת אלגוריתם) — מנחשי-חסם, פיתוח נוסחאות
  צעד-צעד, עצי-עלות נבנים, ומחשבון אינטראקטיבי לשיטת האב.
- **קישורים עמוקים** (deep-links) לכל שיעור/מצב/לשונית, ואתר סטטי ניתן לשיתוף.

## 🏗️ ארכיטקטורה

המערכת בנויה כ**פלטפורמה רב-קורסית**: ליבה אגנוסטית לקורס (`src/core/`) + קורס עצמאי לכל נושא
(`src/courses/<course>/`), עם תפריט קורסים וניתוב `/c/<course>/lecture/<id>/<mode>`. כל קורס נטען
עצלאנית (chunk נפרד), וכתובות ישנות (`/lecture/...`, `/overview`) מופנות אוטומטית.

הליבה היא **מנוע פריימים**: כל אלגוריתם הוא גנרטור דטרמיניסטי טהור שמפיק `Frame[]`. כל המצבים צורכים
את אותה רשימת פריימים; הזהות נוסעת עם הערך (`elementId`) — זה מה שמאפשר לאנימציית ההחלפה לעבוד.

- `src/core/engine/` — טיפוסים, `FrameBuilder`, חשבון אינדקסים (1-indexed, CLRS).
- `src/core/shell/`, `src/core/modes/`, `src/core/viz/`, `src/core/components/` — מעטפת, נגן, תצוגות
  ומערכת-עיצוב — אגנוסטיים לקורס.
- `src/core/platform/` — תפריט הקורסים, ה-router, ה-`CourseProvider` (טעינה עצלה) ועוזרי-הקישורים.
- `src/courses/algorithms/` — הקורס: `registry.ts`, `lectures/<id>/` (כל שיעור מממש `LectureModule`),
  ו-`overview/` (טבלת ההשוואה + המירוץ — ספציפיים למיון).

**הוספת קורס** = ליצור `src/courses/<id>/` שמייצא `CourseModule`, ולהוסיף מניפסט + loader עצל
ב-`src/core/platform/courses.ts`. **הוספת שיעור** = ליצור `src/courses/<course>/lectures/<id>/`
שמייצא `LectureModule`, ולהוסיף שורה ב-registry של הקורס. כל השאר (מצבים, נגן, RTL, deep-links) מגיע חינם.

## ☁️ פריסה

האתר נפרס אוטומטית ל-**GitHub Pages** דרך GitHub Actions בכל `push` ל-`main`
(ראו `.github/workflows/deploy.yml`): התקנה → בדיקות → בנייה → פרסום, עם נפילה ל-`404.html`
כדי ש-deep-links ישרדו רענון. (קיימים גם `vercel.json` ו-`public/_redirects` לפריסה ב-Vercel/Netlify.)
